# Pomodoro-desafio

Projeto de Pomodoro construído a partir do desafio do Programa Explorer, Stage 05.

Tecnologias utilizadas: 
* HTML
* CSS
* JavaScript

No JavaScript, foi posto em prática os conhecimentos sobre:
* ES6 modules
* Event-driven
* Callback
* Padrão Factory e Injeção de dependências
